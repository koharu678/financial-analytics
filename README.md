# 📊 财务自助分析看板

> SQL-Driven Financial Self-Service Analytics

## 🎯 这是什么？

一个**财务自助分析看板**，上传 CSV/Excel 数据后自动生成：

- 📊 KPI 摘要卡片（总收入、总成本、毛利率、费用率/净利率）
- 📈 月度收入/成本趋势图
- 📊 分类毛利率对比图
- 💡 假设分析滑块（敏感性分析）
- 📋 数据明细表（搜索、排序、分页、导出）

**核心叙事**：财务每周花 2 小时手工出表 → 业务部门拖拽上传即分析

## 🔗 访问方式

> 👉 **https://koharu678.github.io/financial-analytics/**

（部署后生效）

## 🖱️ 怎么用？

1. 打开网页 → 看到上传区
2. 拖拽 CSV/Excel 文件到虚线框，或点击"加载模拟数据"
3. 工具自动识别列字段 → KPI 卡片 + 图表 + 表格全部自动生成
4. 拖动假设分析滑块 → 实时看价格变动对利润的影响

## 📁 支持的文件格式

- `.csv` — 拖拽上传
- `.xlsx` / `.xls` — 拖拽上传（读取第一个 Sheet）
- 内置模拟数据 — 点击按钮一键体验

## 🛠 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 | 页面结构和样式 |
| JavaScript (Vanilla) | 所有交互逻辑 |
| Chart.js (CDN) | 趋势图 + 对比图 |
| PapaParse (CDN) | CSV 解析 |
| SheetJS/xlsx (CDN) | Excel 解析 |
| GitHub Pages | 免费部署上线 |
